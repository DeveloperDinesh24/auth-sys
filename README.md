# Supabase Auth Mastery

> **The ultimate authentication boilerplate for modern web apps.** Supabase Auth Mastery is a rock-solid, production-ready starter kit that handles the most complex part of any app: getting users in and out securely.

---

### 🛠 Tech Stack

* **Framework:** React.js + TypeScript
* **State Management:** Zustand
* **Authentication:** Supabase Auth
* **OAuth Integration:** Google & GitHub Providers
* **Styling:** Tailwind CSS

---

### ✨ Key Features

* **Multi-Provider Auth:** Seamless integration for Email/Password, Google, and GitHub login flows.
* **Persistent Session Sync:** Automatic user session recovery on page reload via `onAuthStateChange`.
* **Global Auth Store:** Centralized user state management using Zustand for zero-prop-drilling.
* **Protected Routing:** Intelligent route guarding to prevent unauthorized access to private pages.
* **Modern Error Handling:** Robust `try/catch` logic with real-time feedback for failed login attempts.
* **Secure Logouts:** Full system sign-out that clears both the database session and browser storage.
* **TypeScript First:** Fully typed interfaces for users, sessions, and auth responses.
